# DESAIN SISTEM ABSENSI DIGITAL SMKS RAJASA

# 1. Overview Sistem

Sistem absensi digital berbasis:

* Laravel 12 sebagai backend REST API
* React Native Expo sebagai frontend
* MySQL sebagai database

Sistem dapat berjalan di:

* Android
* iOS
* Web

Menggunakan satu codebase frontend.

Tujuan sistem:

* Digitalisasi absensi sekolah
* Monitoring kehadiran realtime
* Rekap otomatis
* Mengurangi manipulasi absensi
* Mempermudah pihak sekolah

# 12. Relasi Database

users

* belongsTo roles
* hasOne students
* hasOne teachers

students

* belongsTo classes
* belongsTo users
* hasMany attendances

teachers

* belongsTo users
* hasMany schedules

classes

* belongsTo majors
* hasMany students

attendances

* belongsTo students
* belongsTo attendance_sessions

# 13. Sistem QR Code

# Flow

1. Guru membuka session absensi
2. Laravel generate QR token
3. Siswa scan QR via Expo app
4. API validasi:

   * token
   * waktu
   * lokasi
   * status absensi
5. Simpan data attendance

---

# QR Berisi

* session_id
* timestamp
* secure token

---

# Validasi

* Belum absen
* Dalam radius sekolah
* Session aktif
* Tidak melewati jam
